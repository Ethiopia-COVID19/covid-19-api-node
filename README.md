# Ethiopia COVID-19 Response API

Start

```
npm start
```

## Endpoints

Params

- `type`: Type of API (example: `community`, `community-inspections`, `travellers`)
- `id`: (optional) ID of the record pulling

```
http://localhost:3000/v1/:type/:id
```

Example

```
http://localhost:3000/v1/community/1
```
